# domain availability checker (on wedos.com)
check if a domain name is available under these TLDs

* cz 
* eu 
* com 
* net 
* org 
* info 
* biz 
* name 
* pl


## usage:
```
user@host:~# chmod +x ./testurls.sh
user@host:~# ./checkDomainAvailability.sh example
cz yes
eu yes
com no
net no
org no
info no
biz no
pl yes
```
